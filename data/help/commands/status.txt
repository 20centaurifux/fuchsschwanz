USAGE
     /status {flags}
     /status

DESCRIPTION
     Sets visibility, volume and control flags.

     The following flags are available:

          Flag  Name
          ----------------
          v     visible
          s     secret
          i     invisible

          p     public
          m     moderated
          r     restricted
          c     controlled

          q     quiet
          n     normal
          l     loud

     With no argument the status is displayed.
